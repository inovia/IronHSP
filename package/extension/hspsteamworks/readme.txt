・概要

	このアーカイブには、HSPSWプラグインが含まれています。
	HSP3.4以降とともに使用することで、HSPの機能を拡張することができます。
	詳しくは、doclibフォルダ内に含まれているマニュアルをお読み下さい。


・インストール方法

	HSP3.4以降がインストールされているディレクトリ(C:\hsp35等)に
	アーカイブの内容を上書きでコピーしてください。
	(同名のフォルダにそれぞれファイルが追加されるはずです)
	ファイルがコピーされれば、使用するための準備が整います。
