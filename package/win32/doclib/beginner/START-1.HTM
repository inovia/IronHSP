step1：作業フォルダを確認しよう

まず最初は、自分で作業をするためのフォルダを開いてみましょう。HSPでは、最初に扱うためのフォルダを「マイドキュメント」としています。「マイドキュメント」は、スタートメニューやエクスプローラーから開くことができます。他で使用しているファイルがあるかもしれませんが、それはかまいません。 まずはこの入門で必要となるファイルをコピーしましょう。


HSP3TTL.JPG : HSPデモで使われた画像ファイル

このファイルは、HSPをインストールしたフォルダの下にある、 「doclib」の中に置かれています。 「HSP3TTL.JPG」のファイルが見つかったら、最初に開いた 「マイドキュメント」の中にコピーしましょう。 (ファイルの移動をしないように注意してください。) さあ、これで作業フォルダの準備はオーケーです。 ここまでは、まだまだ準備段階。
