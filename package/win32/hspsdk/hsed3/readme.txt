--------------------------------------------------------------------------------
 Hot Soup Processor スクリプトエディタ 外部ツール API
--------------------------------------------------------------------------------

・はじめに

このアーカイブには、Hot Soup Processorのスクリプト エディタとの連携を
行うための方法を記したマニュアルと、簡単に操作できるようにするSDKが
含まれています。



・このアーカイブについて

	docs フォルダ

		ドキュメントをまとめて入れてあります。

		basic.txt … 基本的なエディタ操作方法を書いてありますが、
			SDKが準備されている環境では読む必要ありません。

		以下は各環境用のSDKのマニュアルです。
		hsp3.txt … HSP3用のSDKのマニュアルです。

	sample フォルダ

		SDKを実際に用いたサンプルが含まれます。

		hsp3 フォルダ … HSP3用のサンプルです。
