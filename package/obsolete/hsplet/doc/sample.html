サンプル

基本

mes や boxf などの基本的な命令で、とりあえず動くことを確認してみるサンプルです。

ネットワーク

hspsockA を使用してサーバーに接続するサンプルです。 ただし、ローカルの HTML からサーバーに接続することは出来ないので、上記リンクをクリックしても正しく動作しません。

コンパイラの「テスト実行」をクリックするか、サーバーにアップロードしてから実行する必要があります。

サウンド

midi ファイルを再生してみるサンプルです。

クッキー

hspini を使用してクッキーの読み書きを行うサンプルです。

メニュー

mod_menu.as を使用してメニューを作成するサンプルです。

パスワード

letutil.hsp の passwordInput 命令を使用してパスワード入力欄を作成するサンプルです。

sendmsg

sendmsg 命令を使用してテキストボックスを操作するサンプルです。

デモプログラム

ルービックキューブ

ルービックキューブを遊ぶデモプログラムです。

マウスおよびカーソルキーで操作します。 完成しても何も出ません。

Net-Ball

HSPコンテスト作品 Net-Ball を HSPLet に移植したものです。

マウスで操作します。

ナノ花

HSPコンテスト作品 ナノ花 を HSPLet に移植したものです。

将棋(暫定版)

対戦型将棋です。

マウスで操作します。
