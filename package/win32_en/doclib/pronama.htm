Professional student-chan Professional student-chan collaboration material

Introduction

Professional student material is a programming live broadcast character "Professional student" This material was created in collaboration with the HSP3 mascot character "Tamaon-chan" . Please enjoy the co-starring of Tamane-chan and professional students.



2D material data details

Tamane-chan and Prosei-chan 2D materials are in the "sample / pronama2d" and "hsptv" folders in the HSP installation folder. It contains.
A png format file is available. The background is filled with a color (black) that is transparent with standard HSP. It also contains an alpha channel value that is transparent in HSP3Dish.

"Mini_kei.png" and "mini_tamane.png" are the character images of "Mini Mini Kei-chan" and "Mini Mini Tamane-chan". A total of 15 types (+ character shadows) of fantasy-style battle poses (5 types of facial expressions x 2) drawn with 256 x 256 dots and poses of regular clothes (5 types of facial expressions) are lined up.








The "sample / pronama2d" folder also contains large images. 15 kinds of variations are prepared for each file for both Tamane-chan and Professional Student-chan.
"Mini_chr.hsp" is a sample using the image of 256 x 256 dots in the "HSPTV" folder. Also, "big_chr.hsp" is a sample using a large image.
It can be used in various situations depending on the idea. Sample scripts for displaying on HSP3 are also available, so please use them in your own programs and games.
